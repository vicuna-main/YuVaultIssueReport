### 🧩 插件简介
**YuVault** 是一款围绕“虚拟保险柜 / 跨服仓库”打造的 Bukkit 插件，支持玩家多保险柜、GUI 管理、SQLite 单服存储、MySQL 跨服共享、权限动态容量、世界黑白名单、物品黑白名单、物品回收箱、自动保存、配置热重载等功能。它适合生存服、RPG 服、模组服、多子服群组服，以及需要跨服共享背包扩展或会员仓库容量体系的服务器使用。

### ✨ 核心能力
- 多保险柜管理，支持 `/yvault`、`/yv`、`/vault`
- 可配置 GUI 保险柜列表与回收物品界面
- SQLite / MySQL 存储，适配单服与跨服共享场景
- 权限控制保险柜数量与槽位容量
- 支持管理员打开、创建、删除、调整玩家保险柜
- 世界黑白名单、物品黑白名单与回收箱机制，降低吞物风险
- 自动保存、配置更新、消息文件自定义

### 🛠️ 兼容性说明
插件旨在适配 **Minecraft 1.8 - 1.26** 的多种服务端核心与版本，包括但不限于：

- Paper
- Spigot
- Paper 衍生核心
- Youer
- Arclight
- Mohist
- 其他 Bukkit / Spigot / Paper 系服务端核心

### 🧩 Plugin Introduction
**YuVault** is a Bukkit virtual vault plugin built around player storage and cross-server shared vaults. It supports multiple player vaults, GUI management, SQLite single-server storage, MySQL cross-server sharing, permission-based capacity, world and item access control, reclaimed items, auto-save, hot reload, and configurable messages. It is suitable for survival, RPG, modded, and proxy network servers.

### ✨ Core Features
- Multiple player vaults with `/yvault`, `/yv`, and `/vault`
- Configurable vault list GUI and reclaimed item GUI
- SQLite / MySQL storage for single-server and cross-server use
- Permission-based vault count and slot capacity
- Admin tools for opening, creating, deleting, and resizing player vaults
- World and item access control with reclaimed item handling
- Auto-save, config updates, and customizable messages

### 🛠️ Compatibility
The plugin aims to support **Minecraft 1.8 - 1.26** across multiple server cores and versions, including but not limited to:

- Paper
- Spigot
- Paper forks
- Youer
- Arclight
- Mohist
- Other Bukkit / Spigot / Paper-based server cores

If you encounter compatibility issues, please report them in the QQ group or GitHub Issues and include your server core, Minecraft version, plugin version, error logs, and reproduction steps whenever possible.
